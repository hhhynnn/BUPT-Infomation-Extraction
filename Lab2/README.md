# 环境要求

## 安装[Nodejs](https://nodejs.org/en/)

当前版本是 v16.15.0

```shell
node -v
```

## 安装包管理工具

```shell
npm install -g yarn
```

## 安装后端第三方包

```shell
cd Lab2
yarn
```

## 安装前端第三方包

```shell
cd Lab2/app
yarn
```

## 编译前端静态网站

```shell
cd Lab2/app
yarn build
```

# 运行

## 启动服务器

```shell
cd Lab2
node main.js
```

## 打开浏览器

```shell
chrome http://localhost:8080/
```
